Install on any PC

  1. Download PrivacyMonitor.exe (or the ZIP) to your computer.
  2. Run PrivacyMonitor.exe. No installer — just double-click. First start may take a few seconds.
  3. If Windows shows “Windows protected your PC” or “This app might harm your device”: the app is not code-signed, so Windows shows this for any unsigned exe. Click “More info”, then “Run anyway”. You can verify the file with the SHA256 hash below.
  4. If you see a message about WebView2, install the runtime once (many PCs already have it):
    Download WebView2 Runtime (Microsoft)

Your settings are stored under %LocalAppData%\PrivacyMonitor. The app does not send data to any server.

System requirements

  • Windows 10 or 11 (64-bit)
  • WebView2 Runtime (usually pre-installed with Windows or Edge)
  • No .NET or other frameworks required — the EXE is self-contained
