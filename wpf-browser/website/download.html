If the app doesn’t open after download

Windows often blocks unsigned programs downloaded from the web. Do this before running the EXE:

  1. Unblock the file: Right-click PrivacyMonitor.exe → Properties → at the bottom, check "Unblock" → OK.
  2. Double-click to run. If Windows shows "Windows protected your PC", click "More info" → "Run anyway".
  3. If you see a message about WebView2, install it once: Download WebView2 (Microsoft).
  4. If nothing happens at all: open Command Prompt, run cd /d "C:\Users\YourName\Downloads" (or the folder where the EXE is), then PrivacyMonitor.exe. Any error will appear in the window.

Install on any PC

  1. Download PrivacyMonitor.exe to your computer.
  2. Unblock the file (Right-click → Properties → check "Unblock" → OK).
  3. Run PrivacyMonitor.exe. First start may take a few seconds.

Your settings are stored under %LocalAppData%\PrivacyMonitor. The app does not send data to any server.

System requirements

  • Windows 10 or 11 (64-bit)
  • WebView2 Runtime (usually pre-installed with Windows or Edge)
  • No .NET or other frameworks required — the EXE is self-contained
